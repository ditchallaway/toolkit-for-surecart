== Description ==

This plugin brings together essential admin tools for SureCart store owners, including direct FluentCRM integration, user switching, external products, vacation mode, custom admin columns, customer dashboard customizer, download lists, product visibility overrides, and many more miscellaneous tools.

= Installation =

* Unzip the downloaded zip file.
* Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
* Activate `Toolkit For SureCart` from Plugins page

= Basic features =

* Direct FluentCRM Integration For SureCart
* Trigger FluentCRM Actions Based On SureCart Order, Subscription, And Refund Events
* Check Conditions In FluentCRM Automation Based On SureCart Products, Prices, And Variants
* Create Dynamic FluentCRM Segments With SureCart Purchase Criteria
* Sync SureCart Customers With FluentCRM Profiles
* Add User Switching Links And Buttons To Various SureCart And SurelyWP Addon Related Admin Areas
* View Any Page On Your Site As the Switched User
* Display Specific Product Prices And Variant Prices Anywhere With Shortcodes
* Rename Customer Dashboard Menu Text When Logged Out
* Display The SureCart Menu In The WordPress Admin Bar
* Show A Link To Open The SureCart App In The WordPress Admin Bar

Toolkit For SureCart add a new submenu called "Toolkit" under the "SurelyWP" menu. Here you are able to configure all the plugin settings.
